## How to export the dashboards

In Bitergia:

* Goto "Dashboards Management" -> Saved Objects
* Select the dashboard you want to export
* Click on "Export" button
* The dashboard will be downloaded as a `export.ndjson` file
* Rename the file to something meaningful and add it to this directory
